Design Decisions

Sidebar → Horizontal tabs
De desktop sidebar past niet op mobiel. App-switching wordt een horizontale pill-bar die scrollbaar is. De sidebar chat history wordt bereikbaar via een drawer (swipe right of hamburger menu).
Context panel → Bottom sheet
Het rechter context panel wordt een horizontaal scrollbare card-strip onderaan. Swipe-up gesture om het full-screen te maken (future). Cards behouden dezelfde data, maar zijn compacter.
Chat nav — compact mode
Tijdens actieve chat wordt de nav compacter: back-arrow, brand, en active domain badge. De grid-knop rechts opent de context cards als overlay.
Progressive disclosure
Home toont contextual prompts (net als desktop). Domain modes tonen chips + context cards. Active chat verbergt alles behalve de conversatie. Apple-level focus op de huidige taak.
Touch targets
Alle interactieve elementen ≥ 44pt. Prompt items hebben extra padding voor duim-bereikbaarheid. Input bar respecteert safe-area-inset-bottom.
Visuele consistentie
Zelfde kleurpalet, border-radius, opacity levels als desktop. Geen aparte "mobile look" — het voelt als dezelfde app, alleen geoptimaliseerd voor touch en smaller viewport.
