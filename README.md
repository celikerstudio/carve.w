# Carve Wiki

Evidence-based fitness encyclopedia with 97+ articles on nutrition, exercise science, physiology, training methods, psychology, and injury prevention.

**[Live site →](https://carve-wiki-info-celikernls-projects.vercel.app)**

## About

Carve Wiki organizes fitness knowledge into six domains — all backed by scientific research. Features AI-powered content generation and a search interface built for quick reference.

## Built with

- **Next.js 16** + **React 19** — App Router, server components
- **Supabase** — Database & auth
- **Tailwind CSS** — Styling
- **Vercel AI SDK** — AI-powered content features
- **Mapbox** — Location features
- **Recharts** — Data visualization
